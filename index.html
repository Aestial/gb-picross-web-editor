■ GB Picross Editor ■

Edit .gbsres files

How to use:

  1. Upload a .gbsres file using the "Load File" button
  2. Select grid size (3x3, 5x5, or 10x10)
  3. Click on grid cells to toggle their state (active/inactive)
  4. Download the modified file with the "Save File" button
